# Smart Expense Tracker

A modern, responsive Expense Tracker built with React, Tailwind CSS, and Recharts that helps you track, visualize, and manage your daily expenses efficiently â€” with persistent storage using localStorage.

# Features

* Add, view, and delete expense entries.
* Filter and search expenses by date, amount, or category.
* View total and category-wise spending breakdown.
* Visualize data using charts.
* Use localStorage to persist all expense data.

# Tech Stack

* React JS
* Tailwind CSS
* Recharts(for visualizations)
* Font Awesome
* localStorage(for data persistence)

# Install dependencies

* npm install
  
# Start the development server

* npm start

# App will run at:

* http://localhost:3000

# Charts

* Powered by Recharts:
* Toggle between Pie Chart and Bar Chart
* Dynamic category colors
* Auto recalculation on add/delete/filter
